# แบบฝึกหัด EP4 — แผนที่ข้อมูลเซนเซอร์ IoT

## ส่วนที่ 1: ตรวจสอบอุณหภูมิห้องเครื่องจักร

ใช้ `image_and_contour.py` เป็นจุดเริ่มต้น:

1. ย้ายตำแหน่งเครื่องจักรไปที่ (6, 5)
2. ย้ายเครื่องปรับอากาศไปที่ (1, 1)
3. เพิ่ม sensor node ใกล้จุดร้อนอีกสองตัว
4. กำหนดเส้นเตือนภัยที่ 30 °C ให้เป็นสีแดง
5. อธิบายว่าควรติดตั้งเซนเซอร์เพิ่มบริเวณใด

## ส่วนที่ 2: จำลองปัญหาของข้อมูลจริง

1. เพิ่ม noise ให้เซนเซอร์แต่ละตัว
2. กำหนดให้เซนเซอร์ T3 และ T8 offline
3. แสดงอุปกรณ์ offline เป็นกากบาทสีเทา
4. เพิ่มข้อความแสดงเวลาอัปเดตล่าสุด
5. เปรียบเทียบผลเมื่อใช้ช่วงสีแบบอัตโนมัติกับช่วงคงที่ 22–34 °C

## ส่วนที่ 3: เปลี่ยนชนิดของเซนเซอร์

เลือกหนึ่งตัวแปร:

- ความชื้นสัมพัทธ์ (%RH)
- PM2.5 (µg/m³)
- CO₂ (ppm)
- ความชื้นดิน (%)
- RSSI (dBm)
- แรงสั่นสะเทือน RMS (mm/s หรือ g)

สร้างฟังก์ชันจำลองใหม่ เลือก colormap และกำหนด threshold ที่สอดคล้องกับความหมายของข้อมูล

## โจทย์ท้าทาย: IoT Spatial Dashboard

สร้าง Dashboard ที่มี:

- แผนที่สี
- เส้น threshold
- ตำแหน่งและรหัสอุปกรณ์
- ค่าที่วัดได้พร้อมหน่วย
- สถานะ online/offline
- เวลาอัปเดตล่าสุด
- คำอธิบายว่าบริเวณใดเป็นค่าจริงหรือค่าจาก interpolation
