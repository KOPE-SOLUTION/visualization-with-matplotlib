# EP4 — กราฟเส้นชั้นค่าและแผนภาพสีด้วย Matplotlib

ในบทนี้เราจะนำข้อมูลที่มีตัวแปรอิสระสองตัวและค่าผลลัพธ์อีกหนึ่งตัวมาแสดงบนพื้นที่สองมิติ ตัวอย่างเช่น อุณหภูมิตามตำแหน่ง ความสูงของภูมิประเทศ หรือความเข้มของสัญญาณ แทนที่จะสร้างกราฟสามมิติ เราจะใช้เส้นและสีช่วยสื่อว่าค่าในแต่ละบริเวณสูงหรือต่ำเพียงใด

> เนื้อหา ตัวอย่าง ฟังก์ชันทดลอง และแบบฝึกหัดในบทนี้เรียบเรียงขึ้นใหม่สำหรับหลักสูตรนี้

## วัตถุประสงค์การเรียนรู้

เมื่อเรียนจบบทนี้ คุณจะสามารถ:
- เตรียมกริดสองมิติด้วย `np.meshgrid()`
- วาดเส้นชั้นค่าด้วย `ax.contour()`
- ระบายสีพื้นที่ระหว่างระดับด้วย `ax.contourf()`
- แสดงเมทริกซ์เป็นภาพด้วย `ax.imshow()`
- ใช้ colormap และ colorbar เพื่ออธิบายค่าของข้อมูล
- เลือกวิธีแสดงผลให้เหมาะกับชนิดของข้อมูล
- ซ้อนภาพสี เส้นชั้นค่า และป้ายกำกับไว้ในกราฟเดียว

## 1. ทำความเข้าใจข้อมูลแบบ X, Y และ Z

ข้อมูลที่ใช้ในบทนี้มีองค์ประกอบสามส่วน:
- **X** ระบุตำแหน่งตามแนวนอน
- **Y** ระบุตำแหน่งตามแนวตั้ง
- **Z** คือค่าที่วัดหรือคำนวณได้ ณ ตำแหน่ง (X, Y)

กราฟยังคงมีเพียงสองแกน แต่ใช้เส้นหรือสีแทนค่า Z จึงเหมาะกับข้อมูลภาคสนาม เช่น แผนที่อุณหภูมิ ความกดอากาศ หรือระดับความสูง

<br>

เราจะสร้างสนามข้อมูลจำลองที่มีเนินหนึ่งจุด แอ่งหนึ่งจุด และคลื่นขนาดเล็ก:

~~~python
import numpy as np
import matplotlib.pyplot as plt


def make_field(x, y):
    hill = np.exp(-((x + 1.0) ** 2 + (y - 0.8) ** 2))
    valley = 0.8 * np.exp(-((x - 1.2) ** 2 + (y + 0.7) ** 2) / 0.7)
    ripple = 0.15 * np.sin(2 * x) * np.cos(2 * y)
    return hill - valley + ripple
~~~

ฟังก์ชันนี้เป็นเพียงข้อมูลสังเคราะห์ จึงสามารถปรับตำแหน่ง ความกว้าง และความสูงของแต่ละส่วนเพื่อทดลองได้อย่างอิสระ

## 2. สร้างกริดด้วย `np.meshgrid()`

เริ่มจากสร้างพิกัดหนึ่งมิติสำหรับแกน x และ y จากนั้นแปลงให้เป็นตารางพิกัดสองมิติ:

~~~python
x = np.linspace(-3, 3, 160)
y = np.linspace(-3, 3, 140)

X, Y = np.meshgrid(x, y)
Z = make_field(X, Y)

print(X.shape, Y.shape, Z.shape)
~~~

ทั้ง X, Y และ Z ต้องมี shape เดียวกัน ในตัวอย่างนี้แต่ละสมาชิกของ Z จะตรงกับตำแหน่งหนึ่งตำแหน่งบนกริด

## 3. วาดเส้นชั้นค่าด้วย `contour()`

เส้นชั้นค่าคือเส้นที่เชื่อมตำแหน่งซึ่งมีค่า Z เท่ากัน แนวคิดนี้คล้ายเส้นระดับความสูงบนแผนที่ภูมิประเทศ

~~~python
fig, ax = plt.subplots(figsize=(7, 6))

lines = ax.contour(X, Y, Z, levels=12, colors="black", linewidths=0.8)
ax.clabel(lines, inline=True, fontsize=8, fmt="%.2f")

ax.set(
    title="Contour lines",
    xlabel="x position",
    ylabel="y position",
    aspect="equal"
)
plt.show()
~~~

พารามิเตอร์ `levels` กำหนดจำนวนหรือค่าของระดับที่ต้องการวาด หากส่งเป็นจำนวนเต็ม Matplotlib จะช่วยเลือกระดับที่เหมาะสมให้โดยอัตโนมัติ

`ax.clabel()` ใช้ใส่ตัวเลขลงบนเส้น ทำให้ผู้อ่านทราบค่าของแต่ละระดับโดยไม่ต้องอาศัยสี

## 4. ระบายสีพื้นที่ด้วย `contourf()`

เมื่อมีเส้นจำนวนมาก การระบายสีพื้นที่ระหว่างระดับอาจช่วยให้เห็นโครงสร้างของข้อมูลได้เร็วขึ้น:

~~~python
limit = np.max(np.abs(Z))
levels = np.linspace(-limit, limit, 17)

fig, ax = plt.subplots(figsize=(7, 6))
filled = ax.contourf(X, Y, Z, levels=levels, cmap="coolwarm")

colorbar = fig.colorbar(filled, ax=ax)
colorbar.set_label("Field value")

ax.set(
    title="Filled contour",
    xlabel="x position",
    ylabel="y position",
    aspect="equal"
)
plt.show()
~~~

ตัวอย่างนี้ใช้ช่วงระดับที่สมมาตรรอบศูนย์ร่วมกับ colormap แบบ diverging สีหนึ่งจึงแทนค่าบวก อีกสีแทนค่าลบ และบริเวณกลางแทนค่าที่อยู่ใกล้ศูนย์

## 5. แสดงข้อมูลเป็นภาพด้วย `imshow()`

`imshow()` แสดงสมาชิกในอาร์เรย์สองมิติเป็นพิกเซล เหมาะกับข้อมูลที่อยู่บนกริดสม่ำเสมอและต้องการการแสดงผลที่ต่อเนื่องรวดเร็ว

~~~python
fig, ax = plt.subplots(figsize=(7, 6))

image = ax.imshow(
    Z,
    extent=[x.min(), x.max(), y.min(), y.max()],
    origin="lower",
    cmap="coolwarm",
    aspect="equal"
)

fig.colorbar(image, ax=ax, label="Field value")
ax.set(
    title="Field shown as an image",
    xlabel="x position",
    ylabel="y position"
)
plt.show()
~~~

จุดที่ควรระวัง:

- `imshow()` รับอาร์เรย์ Z โดยตรง ไม่ได้รับกริด X และ Y
- `extent` กำหนดว่าขอบภาพตรงกับค่าพิกัดใด
- ค่าเริ่มต้นของจุดกำเนิดภาพอยู่ด้านบน จึงมักกำหนด `origin="lower"` สำหรับข้อมูลพิกัดคาร์ทีเซียน
- `aspect="equal"` ทำให้หนึ่งหน่วยบนแกน x และ y มีขนาดเท่ากัน

## 6. ซ้อนภาพสีและเส้นชั้นค่า

ภาพสีช่วยให้เห็นภาพรวม ส่วนเส้นชั้นค่าช่วยระบุตำแหน่งของระดับได้แม่นยำ เราสามารถใช้ทั้งสองอย่างร่วมกัน:

~~~python
fig, ax = plt.subplots(figsize=(7, 6))

image = ax.imshow(
    Z,
    extent=[x.min(), x.max(), y.min(), y.max()],
    origin="lower",
    cmap="coolwarm",
    aspect="equal"
)

lines = ax.contour(
    X,
    Y,
    Z,
    levels=[-0.6, -0.3, 0.0, 0.3, 0.6],
    colors="black",
    linewidths=0.8
)
ax.clabel(lines, inline=True, fontsize=8, fmt="%.1f")

fig.colorbar(image, ax=ax, label="Field value")
ax.set(
    title="Image with contour lines",
    xlabel="x position",
    ylabel="y position"
)
plt.show()
~~~

## 7. เลือกใช้ฟังก์ชันไหนดี

| ฟังก์ชัน | เหมาะกับ | จุดเด่น |
|----------|-----------|---------|
| `contour()` | ต้องการอ่านระดับเฉพาะหรือพิมพ์ขาวดำ | เห็นขอบเขตของแต่ละระดับชัด |
| `contourf()` | ต้องการเปรียบเทียบพื้นที่สูง–ต่ำ | อ่านภาพรวมง่ายและควบคุมช่วงระดับได้ |
| `imshow()` | ข้อมูลเป็นเมทริกซ์บนกริดสม่ำเสมอ | แสดงผลรวดเร็วและให้ภาพต่อเนื่อง |
| ใช้ร่วมกัน | ต้องการทั้งภาพรวมและค่าระดับ | สื่อสารรายละเอียดได้ครบที่สุด |

## 8. แนวทางที่แนะนำ

- เลือก colormap ให้สอดคล้องกับความหมายของข้อมูล
- ใช้ colormap แบบ sequential เมื่อค่ามีทิศทางจากน้อยไปมาก
- ใช้ colormap แบบ diverging เมื่อมีค่ากลางที่สำคัญ เช่น ศูนย์
- ใส่หน่วยและคำอธิบายให้ colorbar
- อย่าใช้ระดับมากเกินไปจนเส้นหรือสีรบกวนกัน
- ตรวจสอบ `origin`, `extent` และอัตราส่วนแกนก่อนตีความกราฟ
- หลีกเลี่ยง colormap สีรุ้งเมื่อความสว่างของสีทำให้ลำดับค่าคลาดเคลื่อน
